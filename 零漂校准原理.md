# BMI088 陀螺仪零漂校准原理

## 一、核心算法

### 1.1 迭代公式

零漂校准采用**一阶低通滤波迭代算法**：

```c
// 核心迭代公式
offset_new = offset_old - α × gyro_measured
```

其中：
- `offset`：零漂偏移量（需要估计的值）
- `α`：收敛系数（本项目中为 0.0003）
- `gyro_measured`：当前测量的角速度值

### 1.2 校准公式

校准值应用采用**加法补偿**：

```c
// 校准后输出
output = raw_measurement + offset
```

## 二、算法原理

### 2.1 工作机制

**假设场景**：传感器静止，真实角速度为 0，测量值等于零漂 $b$。

**迭代过程**：
1. 初始状态：`offset = 0`，测量值 `gyro = b`
2. 第一次迭代：`offset = 0 - α × b = -α·b`
3. 第二次迭代：`offset = -α·b - α × (b + (-α·b)) = -α·b(2 - α)`
4. 持续迭代...

**收敛结果**：当迭代次数足够多时，`offset ≈ -b`

**校准效果**：`output = b + (-b) = 0`，实现零漂补偿。

### 2.2 数学分析

该算法本质是一个**一阶低通滤波器**，传递函数为：

$$H(z) = \frac{\alpha}{1 - (1-\alpha)z^{-1}}$$

**时间常数**：$\tau = \frac{1}{\alpha} \cdot T_s$

当采样频率为 1kHz（$T_s = 1ms$），α = 0.0003 时：
- 时间常数 $\tau = 3.3$ 秒
- 约需 3333 次迭代达到收敛

## 三、α 参数的作用

### 3.1 收敛速度控制

| α 值   | 收敛速度            | 稳定性         | 适用场景       |
| ------ | ------------------- | -------------- | -------------- |
| 0.1    | 快（约10次迭代）    | 差，噪声大     | 快速响应       |
| 0.01   | 中等（约100次迭代） | 中等           | 平衡场景       |
| 0.0003 | 慢（约3333次迭代）  | 好，噪声抑制强 | 高精度静态校准 |

### 3.2 α 与响应速度的关系

```
α 增大 → 收敛速度加快 → 噪声抑制能力下降
α 减小 → 收敛速度变慢 → 噪声抑制能力增强
```

## 四、校准流程

### 4.1 流程图

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐
│ 系统启动    │ ──→ │ 加载历史校准值   │ ──→ │ 正常工作    │
└─────────────┘     └─────────────────┘     └──────┬──────┘
                                                   │
                              ┌────────────────────┴────────────────────┐
                              ▼                                         ▼
                       ┌─────────────┐                         ┌─────────────┐
                       │ 触发校准    │                         │ 持续校准    │
                       │ 传感器静止  │                         │ 动态微调    │
                       └──────┬──────┘                         └─────────────┘
                              │
                              ▼
                       ┌─────────────┐
                       │ 迭代计算    │
                       │ offset收敛  │
                       └──────┬──────┘
                              │
                              ▼
                       ┌─────────────┐
                       │ 保存校准值   │
                       │ 写入Flash   │
                       └─────────────┘
```

### 4.2 关键函数

| 函数名                | 功能                      |
| --------------------- | ------------------------- |
| `gyro_offset_calc()`  | 迭代计算零漂偏移量        |
| `INS_cali_gyro()`     | 校准入口，加载历史值      |
| `INS_set_cali_gyro()` | 保存校准值到Flash         |
| `imu_cali_slove()`    | 应用校准（旋转矩阵+零漂） |

## 五、温度补偿

陀螺仪零漂受温度影响显著，项目采用 **PID恒温控制**：

- 目标温度：45°C
- 预热阶段：全功率加热
- 恒温阶段：PID调节加热片PWM
- 效果：减少温度漂移，提高校准稳定性

## 六、总结

| 要点         | 说明                            |
| ------------ | ------------------------------- |
| **算法类型** | 一阶低通滤波迭代                |
| **收敛系数** | α = 0.0003                      |
| **收敛时间** | 约3.3秒（1kHz采样）             |
| **校准方式** | 加法补偿：output = raw + offset |
| **收敛结果** | offset → -b，使静止时output = 0 |
| **温度控制** | 恒温45°C减少热漂移              |
