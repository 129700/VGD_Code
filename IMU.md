### (2)卡尔曼滤波

**目的：减少环境噪声和测量噪声造成的数据偏差**

卡尔曼滤波分为**预测**和**更新**两个阶段，共五个核心公式：

### 阶段一：预测（时间更新）

#### 1. 预测状态（先验估计）
```
x'(k) = F·x(k-1) + B·u(k-1)
```
根据上一时刻的最优估计，预测当前时刻的状态。

#### 2. 预测协方差（先验协方差）
```
P'(k) = F·P(k-1)·F^T + Q
```
`P` 表示状态估计的不确定性（协方差矩阵），`Q` 是过程噪声协方差。

### 阶段二：更新（量测更新）

#### 3. 计算卡尔曼增益
```
K(k) = P'(k)·H^T · (H·P'(k)·H^T + R)^(-1)
```
卡尔曼增益 `K` 决定了预测值和测量值的权重：
- `K` 接近0：更相信预测（测量噪声大）
- `K` 接近1：更相信测量（模型不确定性大）

#### 4. 更新状态（后验估计）

```latex
x(k) = x'(k) + K(k)·(z(k) - H·x'(k))
```
`z(k) - H·x'(k)` 称为**残差**，表示测量值与预测值的差异。

#### 5. 更新协方差（后验协方差）
```
P(k) = (I - K(k)·H)·P'(k)
```
更新后的协方差表示新估计的不确定性。
